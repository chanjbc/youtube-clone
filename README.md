# youtube-clone

This repo contains a demo full-stack YouTube clone for deployment on Google Cloud, and publicly available [here](https://yt-web-client-475659724945.us-east4.run.app/).

## Authors
- [James Chan](https://github.com/chanjbc)

## Features
- Users can sign-in using their Google account
- Signed-in users can upload videos
- Uploaded videos are then transcoded to 360p to save space
- All users can see videos posted to the platform and watch individual videos
